1. Overview — The Full Stack

SchoolConnect is a WhatsApp-first school communication platform. Here is every external service it depends on, grouped by category.

🌿 Stage 1: Initial (0-5 schools)
  • ~500 students total
  • 1 founder running everything
  • Monthly infra cost: ~₹2,100/mo
  • Use free tiers wherever possible
  • Railway Hobby plan
  • Interakt Growth (₹2,499/mo)
🍀 Stage 2: Medium (5-25 schools)
  • ~3,000-5,000 students total
  • Small team (2-3 people)
  • Monthly infra cost: ~₹8,000-12,000/mo
  • Upgrade to paid tiers
  • Railway Pro plan
  • Interakt Advanced (₹3,499/mo)
🍁 Stage 3: Large (25-100+ schools)
  • ~15,000-50,000 students total
  • Full team (5+ people)
  • Monthly infra cost: ~₹25,000-60,000/mo
  • Enterprise plans, dedicated infra
  • Switch BSP to Wati/360dialog
  • Consider direct Meta API

2. Railway — App Hosting & Database

🚂
Railway Paid
railway.com — Cloud platform for deploying apps and databases

What it does

Railway runs your Node.js/TypeScript backend server and your PostgreSQL database. It handles deployment, scaling, environment variables, and networking. Think of it as your entire server infrastructure in one place.

✅ Railway PostgreSQL vs Neon: Use Railway's built-in PostgreSQL. It runs in the same project, same dashboard, same billing. No cold starts. No separate account. For SchoolConnect's scale, Railway PostgreSQL is sufficient and simpler.

💡 Cost tip: Railway bills by actual resource usage per second. If your app is idle at night (no school activity), costs drop significantly. A Node.js app that's 30% idle will cost ~30% less than the 24/7 estimate.

4. Interakt — WhatsApp BSP (Business Solution Provider)

📲
Interakt Required for WhatsApp
interakt.shop — Meta-approved WhatsApp Business API provider

What it does

Interakt is your gateway to WhatsApp Business API. Without a BSP like Interakt, you cannot send WhatsApp messages programmatically. Interakt provides: the API connection to Meta, a dashboard to manage templates, a shared inbox for human agents, chatbot builder, and campaign tools.

⚠ Interakt Starter plan (₹999/mo) is Instagram-only — it does NOT include WhatsApp. The minimum plan for WhatsApp is Growth at ₹2,499/mo.

7. Upstash Redis — Cache & Queue

⚡
Upstash Redis Free to start
upstash.com — Serverless Redis for caching, rate limiting, and queuing

What it does

Redis is an in-memory data store used for three critical functions in SchoolConnect: (1) Rate limiting — prevent parents from spamming the AI assistant, (2) Session caching — store auth tokens and frequently accessed data to reduce DB queries, (3) Message queue — queue WhatsApp messages to send in batches rather than all at once.

8. OpenAI / Gemini — AI Language Model

🤖
OpenAI / Google Gemini Optional but recommended
openai.com / ai.google.dev — Large language models for parent AI assistant

What it does

Powers the AI Parent Assistant — when a parent sends a natural language message like "Why was Rahul absent on Monday?" or "What is the fee due this month?", the LLM understands the question, queries the database, and responds in natural language. WhatsApp inbound messages (service type) are FREE, so the only cost is the LLM compute.

9. Sentry — Error Monitoring

🔍
Sentry Free to start
sentry.io — Application error tracking and performance monitoring

What it does

Sentry automatically captures errors in your Node.js backend and React frontend. When a WhatsApp message fails to send, when a teacher gets a 500 error, or when the attendance API crashes — Sentry captures the full stack trace, the user context, and alerts you immediately. Without Sentry, you find out about bugs when schools complain.
